## Path Configuration

WhiteMagic uses a configurable path system. See `whitemagic/config/paths.py` for details.

**Environment Variables**:
- `WM_STATE_ROOT`: Root directory for state/data (default: `~/.whitemagic`)
- `WM_DB_PATH`: Path to SQLite database
- `WHITEMAGIC_ROOT`: Project root for Mojo components

## Building Components

See the root [SYSTEM_MAP.md](../SYSTEM_MAP.md) for build commands for each component:
- Python: `pip install -e ".[dev]"`
- Rust: `cd whitemagic-rust && maturin develop --release`
- Elixir: `cd elixir && mix compile`
- Haskell: See [Haskell Development](#haskell-development) below

### Haskell Development

The Haskell component requires GHC and cabal. On systems without `libgmp-dev` installed:

```bash
# Create symlink for libgmp (if you have the .so but not -dev package)
mkdir -p ~/.local/lib
ln -s /path/to/libgmp.so ~/.local/lib/libgmp.so

# Build with library path
cd haskell
PATH="$HOME/.ghcup/bin:$PATH" cabal build --ghc-options="-L$HOME/.local/lib"

# Run tests
PATH="$HOME/.ghcup/bin:$PATH" cabal test --ghc-options="-L$HOME/.local/lib"
```

See `haskell/cabal.project.local.example` for a template.
