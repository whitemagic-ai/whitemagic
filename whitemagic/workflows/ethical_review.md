# Workflow: Ethical Review

Review an action through WhiteMagic's full ethical governance stack.

## Steps

1. **Evaluate ethics** — check against Dharma rules
   ```
   gana_straddling_legs(tool="evaluate_ethics", args={"action": "<description>", "context": {}})
   ```

2. **Check boundaries** — verify no boundary violations
   ```
   gana_straddling_legs(tool="check_boundaries", args={"action": "<description>"})
   ```

3. **Review karma** — check recent side-effect audit trail
   ```
   gana_hairy_head(tool="karma_report", args={"limit": 10})
   ```

4. **Check harmony** — get 7-dimension health vector
   ```
   gana_mound(tool="get_yin_yang_balance", args={})
   ```

5. **Governor validation** — final governance check
   ```
   gana_star(tool="governor_validate", args={"action": "<description>", "risk_level": "caution"})
   ```
